## 3. What was built & shipped today
1. **House-of-brands architecture** — new `brands` table + `products.brand` / `products.engine` columns; one Supabase backend now powers many storefronts. Front-end themes per brand via `?brand=` (accent colors, hero copy, logo).
2. **Four niche storefronts launched** — **agentEdge** (real estate · blue), **eComKiller** (ecommerce · red), **bookedBNB** (short-term rental · coral), **coachKit** (coaching · violet). Each carries **3 contentOS + 3 templateVault** kits (24 products total).
3. **Engine provenance** — every brand page splits into a "Made with contentOS" and "Made with templateVault" section, with a provenance badge ("Made with contentOS · by studio0x") on product pages.
4. **Branded product graphics (24/24)** — custom on-brand **SVG illustrations** generated per product and stored in the `product-images` bucket; a floating "kit" card stack on the brand gradient with a category emblem (house / shopping bag / pin+star / growth arrow). No text on the art — our own vector work, fully licensable. Replaces stock placeholders and the old text-heavy covers.
5. **UI redesign (PR #18, Humble-Bundle-inspired)** — **light/dark theme** with a nav toggle (respects OS preference, remembered), **square 1:1 image tiles**, **grid/list** view toggle, cleaner alignment, and **camelCase** product names throughout.
6. **AI generator upgraded** — `ai-create-product` now accepts `brand` + `engine` params and stamps both onto generated products; used it to batch-generate the new catalog.
7. **camelCase brand naming convention** — saved to global memory so every future project/product follows it (lowercase first segment + Capitalized suffix, e.g. agentEdge, coachKit, bookingBoost).
8. **Migrations** — `0006_brands` (brands table + columns), `0007_brand_rename` (final brand names + bold per-brand accents, catalog repointed, stray kits archived), and `fix_names_add_photos` (lowercased all 24 names to convention, added `products.photo_url`).
