## Key Product Decisions

### The stat engine is the product. The World Cup is the use case.
This framing must drive everything going forward. Studio0x is a proprietary AI-powered stat engine for elite sports. WC 2026 is the live reference deployment. The platform is white-label and reskinnable for F1, UCL, NFL. This story is not currently being told on the site — fixing that is tomorrow's priority #4.

### Current site tells the wrong story
- Landing on `/` reads as a scoreboard app, not a stat platform
- Studio0x brand is secondary to "WC 2026"
- White-label / multi-sport angle is invisible
- Formula footnotes on metric cards undercut the proprietary feel

### Timezone: works for free via browser
`toLocaleTimeString()` with no explicit timezone automatically converts UTC to the user's local time. Foreign users get correct match times without any code changes.

### Language: English only, Chrome fills the gap
No i18n library in place. Chrome's built-in translation offers auto-translate for static UI text. Dynamic AI content (commentary, stories, talking points) stays in English regardless. Real i18n deferred — priority markets when ready: Brazilian Portuguese, Spanish.
